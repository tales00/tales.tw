這個頁面…

  • 用 Vue.js 寫的
  • 本網站最佳瀏覽解析度為 1024 x 768 因為有做 RWD ，只要長寬比不太奇怪應該看起來都還不錯
  • 寫不出什麼有料的東西，所以想到把自己平常在撲浪的碎碎念接過來假裝成 Blog
  • 既然如此，身為一個前端工程師，順便把圖從 Pixiv 撈過來也都是很合理的
  • 於是這個頁面的 Blog 與 Illust 的資料都是另外串 API 取得的
  • API 要來的資料有用 Local Storage 做前端快取
  • 不過 Pixiv 沒有提供 API 給人撈資料，所以我是自己寫爬蟲再做成 API 給自己接
  • 後端的爬蟲是用 PHP 寫的，使用一套叫做 Fat-free framework 的 MVC 框架
  • 我也有 Twitter 帳號，但是沒什麼在發言就沒接了
  • 在考慮也寫爬蟲把放在蝦皮上的東西也撈過來充實內容
